Editor de Texto

Lo primero para poder programar es tener las ganas de hacerlo y lo segundo es un editor de texto con el que te sientas comodo para esto tenemos muchas opciones como bien puede ser el Visual Estudio code, el Sublime Text, el Dreamweaver entre otros.

Estructura

La estructura de un documento HTML se puede resumir así:

Dentro de la etiqueta "html" esta basicamente contiene toda el contenido de la pagina desde lo que podemos ver hasta lo que no podemos ver y aqui obtenemos 2 conceptos: lo que podemos y no podemos ver, pero ¿como podemos diferenciar esto?.

Lo que no podemos ver se situa dentro de la etiqueta "head" aqui podemos colocar cosas que no se veran dentro de la pagina como el titulo de la pagina que se muestra en la pestañita arriba del navegador y las demas configuraciones y los enlaces a los estilos.

Lo que podemos se situa en la etiqueta "body>" los parrafos , las imagenes, todo lo que se puede ver dentro de la pagina se guarda aqui

Titulo

Las primeras estiquetas que veremos dentro del body o cuerpo dentro de nuestra pagina Web seran las: "h1,h2,h3,h4,h5,h6" estas van del h1 al h6 y son los distintos tipos de titulos y subtitulos que tengamos dentro de nuestra pagina, lo preferencial es que tengamos solo un h1 para el titulo y el h2 una cierta cantidad de veces y asi con los demas esto es para tener un mejor control dentro de la pagina para no perdernos dentro del codigo

Parrafo

El parrafo esta definido por la etiqueta "p" y como su nombre lo dice nos sirve para escribir parrafos y aunque dentro de nuestra pagina web se ve como otra linea de texto mas como los titulos este es un texto correcto para hacer un parrafo

Imagenes

Para poner imagenes dentro de HTML se usa la etiqueta "img" esta tiene un atributo llamado "src=""" aqui podemos poner imagenes en linea o locales que nosotros tengamos guardadas dentro de nuestra pc tambien tenemos los atributos "widht y eight" que estos nos sirven para cambiar el tamano de la imagen poniendo dentro del atributo el tamaño que queremos que tenga en pixeles.

Enlaces

La siguiente etiqueta que tenemos es la "a" esta etiqueta lo que hace es redirigirnos a otra pagina o subtitulo dentro de nuestra pagina esta tiene el atributo "href=""" que es donde debemos poner el enlace o ruta al que queremos ir, tenemos 2 tipos de rutas: las locales y las externas, las locales son las que tenemos dentro de nuestro servidor y no podemos enlazar localmente y las externas son las que no enlazamos de manera local en estas tenemos que colocar el enlace de la pagina que queremos ir debemos colocar lo siguiente.
